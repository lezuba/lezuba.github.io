Bienvenido a mi espacio

Hola, este es mi nuevo sitio web. Es un lugar sencillo para compartir mis pensamientos e ideas. Puedes modificar este texto directamente en el archivo index.html. La etiqueta <strong> sirve para poner texto en negrita.

Aquí puedes añadir otro párrafo con más información. Por ejemplo, puedes poner un enlace. Para crear un enlace, usas la etiqueta <a>. Por ejemplo, aquí tienes un enlace a la página de GitHub. El atributo 'target="_blank"' hace que el enlace se abra en una nueva pestaña.
